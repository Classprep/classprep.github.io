계정(Account)

클래스프렙 계정을 만드실 때, 실제 존재하고 있는 이메일을 사용하셔야 합니다. 실제 존재하지 않는 이메일을 입력하셨을 경우 가입이 되지 않으니 실제 존재하는 이메일로 다시 회원가입 해주시기 바랍니다.
만약 로그인에 문제가 있다면, 입력하신 이메일이 제대로 입력 되었는지 확인해주세요!
회원가입 때 사용했던 이메일이 생각나지 않는다면, 메인 화면에 이메일 찾기(링크로 갈 수 있게)로 가시면 됩니다.
비밀번호가 생각이 나지 않으신다면, 메인 화면에 비밀번호 찾기(링크로 갈 수 있게) 로 가시면 됩니다.
로그인 하신 후, 우측 상단의 설정 버튼(아이콘)을 누르시면, 타임 존을 변경하실 수 있습니다.
로그인 하신 후, 우측 상단의 설정 버튼(아이콘)을 누르시면, 언어를 변경하실 수 있습니다.
로그인 하신 후, 우측 상단의 설정 버튼(아이콘)을 누르시면, 이름을 변경하실 수 있습니다.
